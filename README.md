# my-cli-commands
a simple cli tool for daily tasks


## build

$ go build -o bin/tt
$ mv ./bin/tt ~/bin/

## commands

```
Usage:
  tt [command]

Available Commands:
  clear       clear files
  hello       say hello
  help        Help about any command
  version     show version

Flags:
  -h, --help   help for tt

Use "tt [command] --help" for more information about a command.
```
